Hello coders!


This online code editor was a project undertaken by the team Web Waltzers for the WIT Project Sprint event organised by GDSC IIEST. Our team consist of four members-- Nayonika Chakraborty, Sushmita Singh, Enakkhi Saha and Ankita Barua. We are currently sophomores in college and aspiring developers, and this simple project helped us to understand and implement whatever we have learnt so far.

An online code editor is a tool that resides on a remote server and is accessible via browsers. For any developer, be it amateur or professional, often the liberty of using a local code editor may be unavailable. As online code-editors are fast, efficient and greatly popular, it is a familiar tool among developers. The tech-stack used in making this project is HTML, CSS and JavaScript. Reason for following this tech-stack is that these languages are easy to use and are also fast in terms of execution time.
